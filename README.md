## 💻 Лабораторные работы

### Лабораторная работа №1: Сегментация текста (Омонимия)

**Тема:** Динамическое программирование, префиксные деревья (боры)

**Задача:**  
Дана строка S и словарь слов с весами. Требуется разбить строку на слова из словаря так, чтобы сумма весов была максимальной.

**Реализация:**

- Префиксное дерево для хранения словаря
- Алгоритм динамического программирования O(n * L_max)
- Визуализация дерева с помощью библиотеки rich

**Режимы работы:**

- `1` - Обычный ввод
- `2` - Стресс-тест
- `3` - Тестирование
- `/help` - Справка

**Пример использования:**
Выберите режим:
[1] Обычный ввод
[2] Стресс-тест
[3] Тестирование
[/help] Справка
→ 1

Введите строку S (≤ 10^5, только a-z): helloworld

Введите количество записей (≤ 10^4): 2
Введите записи в формате: слово вес
Запись 1: hello 5
Запись 2: world 3

Результат:
✅ Максимальная сумма: 8.00
📝 Разбиение:
hello(5.00) + world(3.00)

## ⚙️ Установка и запуск

### Предварительные требования

- Python 3.8 или выше
- pip

### Установка

1. Клонируйте репозиторий:

    ```bash
    git clone https://github.com/yourusername/DP.git
    cd DP
    ```

2. Перейдите в директорию нужной лабораторной работы:

    ```bash
    cd lab1
    ```

3. Установите зависимости:

    ```bash
    pip install -r requirements.txt
    ```

### Запуск

```bash
python main.py
```

### Типы тестов

- Позитивные тесты - корректные данные
- Негативные тесты - проверка обработки ошибок
- Граничные тесты - предельные случаи
- Тесты производительности - проверка скорости
- Интерактивные сценарии - пользовательский ввод
